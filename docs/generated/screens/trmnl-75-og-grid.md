The home screen uses a **4-row x 5-column** grid, giving you **20 card slots**. Any home-screen card can be turned into a [Subpage](/features/subpages) folder containing up to 19 more cards.

Flexible card sizes are supported: Single, Tall, Wide, and Large.

| Capability | Value |

| Built-in relays | No built-in relays |
